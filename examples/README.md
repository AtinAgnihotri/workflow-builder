# Examples

Integration examples for `@journeys/core`. See [docs/examples.md](../docs/examples.md)
for the full index and run commands.

| Directory | Command |
| --- | --- |
| [vanilla](vanilla/) | `pnpm --filter example-vanilla start` |
| [react-local-state](react-local-state/) | `pnpm --filter example-react-local-state dev` |
| [react-router](react-router/) | `pnpm --filter example-react-router dev` |
| [tanstack-router](tanstack-router/) | `pnpm --filter example-tanstack-router dev` |
| [next](next/) | `pnpm --filter example-next dev` |
| [edge-runtime](edge-runtime/) | `pnpm --filter example-edge-runtime start` |

Build core first: `pnpm --filter @journeys/core build`.
